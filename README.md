# Scrapyard Admin Panel

Scrapyard Admin Panel is a web-based dashboard for managing the Scrapyard ecosystem.

## Features

- Admin Authentication
- User Management
- Orders Management
- Product Management
- Refurbished Products Management
- Scrap Requests Management
- Notifications System
- Reviews Management
- Settings Management
- Analytics Dashboard

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Firebase Authentication
- Firestore Database
- Firebase Storage
- Firebase Cloud Messaging
- Firebase Hosting
